## Color Scheme
- **Primary Blue**: Main accent color for headers and metrics
- **Green (#16a34a)**: Scores â‰¥80%, valid status
- **Red (#dc2626)**: Scores <80%, critical/expired status
- **Orange (#f97316)**: Medium severity, warnings
- **Yellow (#eab308)**: In progress status
- **Gray**: Neutral/secondary information

## Component Features

### Interactive Elements
1. **Export Button**: Allows downloading compliance reports
2. **Tab Navigation**: Switch between different views
3. **View Details Buttons**: Access detailed information for each item
4. **Progress Bars**: Visual representation of corrective action completion

### Data Visualization
- **Score Cards**: Large numeric displays with icons
- **Badges**: Status indicators with color coding
- **Progress Bars**: Linear progress indicators for actions
- **Color-Coded Scores**: Visual differentiation of compliance levels

### Responsive Design
- Grid layout adapts to screen size (1-4 columns)
- Cards stack vertically on mobile devices
- Tabs remain accessible across all breakpoints
