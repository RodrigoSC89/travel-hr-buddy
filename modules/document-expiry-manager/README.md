# Document Expiry Manager

**PATCH 655** - Automatic document expiry detection using OCR and LLM

## Overview

Automatic document expiry detection using OCR and LLM

## Features

- OCR extraction
- LLM date parsing
- Automated alerts
- Document tracking

## Usage

```typescript
import DocumentExpiryManager from "@/modules/document-expiry-manager";

// In your route configuration
<Route path="/documents/expiry" element={<DocumentExpiryManager />} />
```

## Technical Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Database**: Supabase
- **UI Components**: shadcn/ui
