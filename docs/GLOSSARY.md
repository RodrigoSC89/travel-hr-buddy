## Technical Terms

### Frontend

| Term | Definition |
|------|------------|
| **Component** | Reusable UI building block in React |
| **Hook** | Function that lets you use React features |
| **Props** | Data passed to components |
| **State** | Component's internal data |
| **Context** | Way to share data across components |

### Backend

| Term | Definition |
|------|------------|
| **Edge Function** | Serverless function running on Supabase |
| **RLS** | Row Level Security - database access control |
| **Migration** | Database schema change script |
| **JWT** | JSON Web Token - authentication token |

### Architecture

| Term | Definition |
|------|------------|
| **SaaS** | Software as a Service |
| **Multi-tenant** | Single app serving multiple organizations |
| **PWA** | Progressive Web App - installable web app |
| **SSR** | Server-Side Rendering |
| **CSR** | Client-Side Rendering |
