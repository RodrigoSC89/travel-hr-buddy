## ðŸ–±ï¸ User Interactions

### Click Actions
1. **Iniciar Button**: 
   - Sends POST to `/api/os/update`
   - Updates status to `em_andamento`
   - Shows success/error toast

2. **Concluir Button**:
   - Sends POST to `/api/os/update`
   - Updates status to `concluido`
   - Shows success/error toast

3. **Exportar PDF Button**:
   - Generates PDF using html2pdf.js
   - Downloads file immediately
   - Shows success/error toast
