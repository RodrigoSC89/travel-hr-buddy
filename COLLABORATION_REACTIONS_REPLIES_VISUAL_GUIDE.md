## UI/UX Features

### Color Scheme
- **Reactions**: Outlined buttons with hover effect
- **Replies**: Light gray background (`bg-gray-50`)
- **Reply Border**: Gray left border (`border-l-2 border-gray-200`)
- **Icons**: Muted foreground color

### Spacing
- **Gap between reactions**: 2 units (`gap-2`)
- **Padding in replies**: 3 units (`p-3`)
- **Left indent for replies**: 4 units (`pl-4`)
- **Space between sections**: 3-4 units (`space-y-3`, `space-y-4`)

### Typography
- **Timestamps**: Extra small, muted (`text-xs text-muted-foreground`)
- **Emails**: Extra small, muted (`text-xs text-muted-foreground`)
- **Reply text**: Small (`text-sm`)
- **Icons**: 3x3 or 4x4 pixels

### Responsive Design
- **Buttons**: Small size for reactions (`size="sm"`)
- **ScrollArea**: Fixed height of 600px
- **Cards**: Responsive padding
- **Layout**: Stack vertically on mobile
