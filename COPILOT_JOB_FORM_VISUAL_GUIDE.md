## Component States

### JobFormWithExamples States
1. **Initial State**: Empty form fields
2. **Typing State**: User typing in description
3. **Filled State**: Both fields have values
4. **Submit State**: User clicks submit button

### SimilarExamples States
1. **Hidden**: When description is empty or < 3 characters
2. **Loading**: During search (shows spinner)
3. **Results**: Displaying similar examples
4. **Empty**: No results found
5. **Error**: Error during search (handled silently)
